Accesibilidad

Declaración de accesibilidad Forma


Forma se compromete a garantizar la accesibilidad digital para personas con discapacidades. Estamos continuamente mejorando la experiencia del usuario para todos y aplicando los estándares de accesibilidad pertinentes.


Estado de conformidad


Las Pautas de Accesibilidad para el Contenido Web (WCAG) definen los requisitos para que los diseñadores y desarrolladores mejoren la accesibilidad para las personas con discapacidades. Define tres niveles de conformidad: Nivel A, Nivel AA y Nivel AAA. Forma cumple parcialmente con WCAG 2.1 nivel AA. Parcialmente conforme significa que algunas partes del contenido no se ajustan completamente al estándar de accesibilidad.
